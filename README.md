# Credit-Card-Fraud-Detection---ML
Credit card fraud is a major problem in the financial industry. Fraudulent transactions can cause huge financial losses for banks and customers. The goal of this project is to build a machine learning model that can detect fraudulent credit card transactions based on transaction data.  This project uses supervised machine learning for legimate 

Problem Statement

The objective of this project is to develop a machine learning model that can accurately identify fraudulent credit card transactions while minimizing false positives.
The dataset is highly imbalanced, meaning fraud transactions are very rare compared to normal transactions, which makes the problem challenging.
